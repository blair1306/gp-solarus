93/**
94 * \brief This class represents a living Lua context that can execute quest
95 * scripts at runtime.
96 *
97 * Such scripts include the quest main script, map scripts, enemy behaviors,
98 * etc. This class implements the Solarus scripting API that scripts can use.
99 *
100 * All these scripts run in the same Lua context. This means that they share
101 * global values and other global mechanisms like the registry and metatables.
102 *
103 * On the contrary, data files that happen to have a Lua-compatible syntax
104 * (like maps) are always parsed in their own, independent Lua states.
105 * These files are considered as pure data (not code) and only use the
106 * LuaTools class, not this class.
107 */
